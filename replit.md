# Project Overview

## Full-Stack JavaScript Application
This is a modern full-stack web application built with:
- **Frontend**: React + TypeScript + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Express + TypeScript + PostgreSQL + Drizzle ORM
